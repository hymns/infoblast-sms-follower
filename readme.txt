== Description ==

Infoblast SMS Follower is a WordPress plugin by Muhammad Hamizi Jaminan (http://www.hamizi.net/) inspired by RSS Feed from Email to SMS, that allows you to place a widget on your blog where readers can follow/unfollow blog updates by receiving SMS Text directly to their mobile phones.

If your blog makes use of user registrations, registered users can register their mobile numbers to receive SMS message updates through their profile page.

Blog owners have the functionality to send out SMS Text messages to follower numbers through the WordPress dashboard and also managing numbers and widget options from there. Contacts can be imported and exported in CSV format and owners have the choice to send SMS messages to just widget follower or registered users or both.

You can now also send out SMS alert to your followers when you post a new blog post.

We accept no responsibility for SMS messages send wrongfully, use at your own risk.

== Installation ==

= Installation =

1. Upload `infoblast-sms-follower` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Basic Setup =

1. Sign up with [Infoblast SMS Provider](http://www.infoblast.com.my)
2. Send email to tarmizi [at] tab [dot] com [dot] my and mention to Activate API account. Please provide your username on your email.
3. Setup account details under the SMS Follower -> Settings menu in your WordPress dashboard.

= Add widget to sidebar =

1. Go to Apearance -> Widgets in your WordPress dashboard
2. Drag the SMS Follower widget to your desired widget area
